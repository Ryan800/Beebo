Beebo
=====

Sina Weibo Client
![image](http://img.wdjimg.com/mms/icon/v1/d/44/52ed35dcd4591c4cdbd5cfccc4b2344d_256_256.png)

-----------------
暂时不支持平板分辨率
-----------------
Google Play 上架的版本叫：Beebo+；

国内市场上线的叫：Beebo；

两者唯一的区别是：Beebo+可智能解析微博中视频链接，直接播放视频，因此apk体积会大一点。
